# 家庭云端相册公众号系统 — 部署操作手册

> 本手册描述从零开始搭建整套系统。按第1步到第5步顺序执行。
> ⚠️ 本手册不含任何账号密码，所有 `<占位符>` 需替换为实际值。

---

## 第1步 准备代码仓库

1. 在 GitHub 创建仓库，clone 项目代码到本地
2. 确认项目文件结构完整：
   ```
   项目根目录/
   ├── server/           # Express + TypeScript 后端
   ├── client/           # React + TailwindCSS 前端
   ├── miniapp/          # 微信小程序壳（可选）
   ├── uploads/          # 图片暂存目录
   ├── data/             # 本地JSON数据目录
   ├── Dockerfile        # Render 部署配置
   ├── render.yaml       # Render 一键配置
   └── 部署指南.md
   ```

---

## 第2步 注册第三方服务

### 2.1 GitHub 仓库
- 创建公开仓库，推送代码（`git push origin main`）

### 2.2 Render (服务托管)
- 注册 [render.com](https://render.com)，用 GitHub 登录
- 连接你的代码仓库，自动识别 Dockerfile
- **Region**: Singapore，**Instance Type**: Free
- 环境变量清单（见第4步）

### 2.3 Cloudinary (图片存储)
- 注册 [cloudinary.com](https://cloudinary.com)，免费 25GB
- 在 Dashboard 获取：Cloud Name、API Key、API Secret

### 2.4 cron-job.org (防休眠)
- 注册 [cron-job.org](https://cron-job.org)
- 创建任务：URL=`https://你的域名.onrender.com/api/health`，每10分钟

### 2.5 微信公众号 (消息接收)
- 用**任一家庭成员**的微信注册个人订阅号（免费）
- ⚠️ 注册人不能自己发消息给公众号（微信限制）
- 注册完成后获取：AppID、AppSecret

---

## 第3步 配置微信公众号

### 3.1 微信开发者平台
- 打开 [developers.weixin.qq.com/platform](https://developers.weixin.qq.com/platform)
- 进入「我的业务 → 公众号 → 基础信息」

### 3.2 域名与消息推送配置
```
URL：              https://你的域名.onrender.com/api/wechat/callback
Token：            <你设定的Token>（可自定）
EncodingAESKey：   点「随机生成」
消息加解密方式：    明文模式
```

### 3.3 IP 白名单
```
74.220.52.251（Render 当前出口 IP，视部署变化自调）
```

### 3.4 JS 接口安全域名
```
你的域名.onrender.com
```
验证文件放于 `client/public/MP_verify_XXXXXXXX.txt`，内容为微信提供的字符串。

---

## 第4步 配置 Render 环境变量

在 Render → Environment 添加以下变量：

| Key | Value |
|---|---|
| `WECHAT_TOKEN` | `<你的Token>` |
| `WECHAT_APP_ID` | `<你的AppID>` |
| `WECHAT_APP_SECRET` | `<你的AppSecret>` |
| `SITE_URL` | `https://你的域名.onrender.com` |
| `ADMIN_USERNAME` | `admin` |
| `ACCESS_PASSWORD` | `<管理员密码>` |
| `CLOUDINARY_CLOUD_NAME` | `<Cloudinary Name>` |
| `CLOUDINARY_API_KEY` | `<Cloudinary API Key>` |
| `CLOUDINARY_API_SECRET` | `<Cloudinary API Secret>` |
| `CLOUDINARY_STORAGE_ALERT_GB` | `22.5` |
| `JOIN_PASSPHRASE` | `<你设定的暗号>` |

保存后 Render 自动重新部署（约2分钟）。

## 第5步 验证

### 5.1 服务器在线
```bash
curl https://你的域名.onrender.com/api/health
# 应返回 {"status":"ok","time":"..."}
```

### 5.2 微信回调通过
在微信开发者平台点「提交」后，显示「提交成功」即回调已通。

### 5.3 测试照片
1. 用另一个微信关注公众号
2. 发送暗号 → 反问昵称 → 回复昵称 → 加入白名单
3. 回复昵称
4. 发送照片 → 应收到确认语句 + 链接
5. 打开网站 → 照片墙应显示照片
