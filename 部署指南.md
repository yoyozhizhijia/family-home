# 🏡 家庭时光 — 部署指南

## 第一步：上传代码到 GitHub

1. 去 [github.com](https://github.com) 注册/登录
2. 新建一个 **私有仓库**（Private），命名为 `family-home`
3. 把项目代码推上去：

```bash
cd "家庭项目"
git init
git add .
git commit -m "家庭时光第一版"
git branch -M main
git remote add origin https://github.com/你的用户名/family-home.git
git push -u origin main
```

## 第二步：Render 免费部署

1. 去 [render.com](https://render.com) 用 GitHub 账号注册/登录
2. 点右上角 **New +** → **Web Service**
3. 连接你的 GitHub，选择 `family-home` 仓库
4. Render 会自动识别根目录的 `Dockerfile`，以下参数确认即可：
   - **Name**: `family-home`（随意）
   - **Region**: 选 `Singapore`（离国内近）
   - **Instance Type**: `Free`
5. 滚动到 **Environment Variables**，填入以下变量：

| 变量名 | 值 |
|---|---|
| `SITE_URL` | 部署后 Render 会告诉你域名，改成 `https://xxx.onrender.com` |
| `WECHAT_TOKEN` | `familyHome2024` |
| `WECHAT_APP_ID` | `wxbfea414dc839abf0` |
| `WECHAT_APP_SECRET` | `e8d9cfdd08012bb43c9ae1d53e1786bb` |
| `ADMIN_USERNAME` | `admin` |
| `ACCESS_PASSWORD` | `1578987313` |

6. 点 **Deploy Web Service**，等 3-5 分钟构建完成
7. 部署成功后，你会得到一个域名：`https://family-home-xxxx.onrender.com`

## 第三步：配置微信公众号

1. 登录 [微信开发者平台](https://developers.weixin.qq.com/platform/)
2. 进入「我的业务」→ 你的公众号 →「基础信息」→「域名与消息推送配置」
3. 启用消息推送，填写：

| 字段 | 值 |
|---|---|
| URL | `https://family-home.onrender.com/api/wechat/callback` |
| Token | `familyHome2024` |
| EncodingAESKey | 点 **随机生成** |
| 消息加解密方式 | 选 **明文模式** |

4. 同一页面的 **API IP 白名单** 加上 `74.220.52.251`
5. **JS 接口安全域名** 加上 `family-home.onrender.com`（验证文件已部署，直接点验证）
6. 保存并提交

## 第四步：添加家庭成员

现在支持暗号自动加入——无需在后台手动添加：

1. 家人关注公众号后，发送 `悠悠之之的大力堡芝麻开门`
2. 公众号反问「请问你是？」，回复昵称即可自动加入
3. 之后发照片自动上墙

管理员仍可在网站上点「管理」→「成员」手动编辑或删除成员。
