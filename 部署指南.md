# 🏡 家庭时光 — 部署指南

## 第一步：上传代码到 GitHub

1. 去 [github.com](https://github.com) 注册/登录
2. 新建一个 **私有仓库**（Private），命名为 `family-home`
3. 把项目代码推上去：

```bash
cd "家庭项目"
git init
git add .
git commit -m "家庭时光第一版"
git branch -M main
git remote add origin https://github.com/你的用户名/family-home.git
git push -u origin main
```

## 第二步：Render 免费部署

1. 去 [render.com](https://render.com) 用 GitHub 账号注册/登录
2. 点右上角 **New +** → **Web Service**
3. 连接你的 GitHub，选择 `family-home` 仓库
4. Render 会自动识别根目录的 `Dockerfile`，以下参数确认即可：
   - **Name**: `family-home`（随意）
   - **Region**: 选 `Singapore`（离国内近）
   - **Instance Type**: `Free`
5. 滚动到 **Environment Variables**，填入以下变量：

| 变量名 | 值 |
|---|---|
| `SITE_URL` | 部署后 Render 会告诉你域名，改成 `https://xxx.onrender.com` |
| `WECHAT_TOKEN` | `familyHome2024` |
| `WECHAT_APP_ID` | `wxb0716a92b8513191` |
| `WECHAT_APP_SECRET` | `c4e12a1030642f6da4f36682bdd68cf2` |
| `ADMIN_USERNAME` | `admin` |
| `ACCESS_PASSWORD` | `family2024`（**务必改成你自己的！**） |

6. 点 **Deploy Web Service**，等 3-5 分钟构建完成
7. 部署成功后，你会得到一个域名：`https://family-home-xxxx.onrender.com`

## 第三步：配置微信公众号

1. 登录 [mp.weixin.qq.com](https://mp.weixin.qq.com)
2. 左侧菜单 → **设置与开发** → **基本配置**
3. 找到「服务器配置」，点 **修改配置**：

| 字段 | 值 |
|---|---|
| URL | `https://family-home-xxxx.onrender.com/api/wechat/callback` |
| Token | `familyHome2024` |
| EncodingAESKey | 点 **随机生成** |
| 消息加解密方式 | 选 **明文模式** |

4. 点 **提交** — 微信会立刻校验，通过即显示「提交成功」

## 第四步：添加家庭成员

1. 让每个家人关注公众号
2. 每人给公众号发一条消息（任意文字即可）
3. 你登录 Render 后台 → family-home → **Logs**，找到类似这行：

```
[微信] 非家庭成员 (oABC123xyz...)，已忽略
```

4. 复制 `oABC123xyz...` 这个 OpenID
5. 访问你的网站 → 点「🔐 管理」→ 用 `admin` / 你的密码登录 → 点「👨‍👩‍👧‍👦 成员」→ 填入 OpenID + 昵称 → 添加
6. 重复以上步骤，把 8 个家人都加进来

## 第五步：设公众号菜单（可选）

在公众号后台「自定义菜单」里添加一个菜单项，类型选「跳转网页」，URL 填你的 Render 域名。这样家人点菜单就能直接进照片墙。

## ⚠️ 免费版 Render 注意事项

- 15 分钟无访问会自动休眠，下次访问需等 30-60 秒唤醒
- 每月 750 小时免费额度（刚好够一个实例 24x7 运行）
- 上传的照片存储在 Render 容器内，**重新部署后会丢失**。建议后续挂载 Render Disk（付费，$1/月起）或改用 Cloudflare R2（免费 10GB）
